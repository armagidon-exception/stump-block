# Stump Block
Convert C# code to Block Diagram

# Supported language constructions
1. Variables
2. Methods
3. If-statements
4. While loops
5. Do-while loops
6. Input through Console.ReadLine()
7. Output through Console.WriteLine()

# Supported blocks for rendering
1. Box
2. Subroutine
3. Input / Output
4. Branching
